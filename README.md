### Run Locally

To run **Dyness** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Dyness Abeid/Dyness-portfolio.git
```

Windows:

```bash
git clone https://github.com/Dyness Abeid/Dyness-portfolio.git
```
